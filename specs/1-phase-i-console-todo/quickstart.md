# Quickstart: Phase I Console Todo

## Setup
1. Ensure Python 3.13+ is installed.
2. Initialize with UV (optional): `uv init`.

## Running the Application
```bash
python phase-i/src/main.py
```

## Available Commands
- `add <title>`: Add a new task.
- `list`: View all tasks.
- `complete <ID>`: Toggle task completion.
- `update <ID> <new_title>`: Change task title.
- `delete <ID>`: Remove a task.
- `help`: Show this menu.
- `quit`: Exit the app.
